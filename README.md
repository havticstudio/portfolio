# Portfolio

A modern, responsive developer portfolio website built with Next.js 15, React, and Tailwind CSS.

## Features

- **Work Experience**: Interactive timeline of professional experience.
- **Expert Area**: Showcasing skills in React, Next.js, JavaScript, Node.js, Tailwind CSS, and HTML.
- **Recent Projects**: Displays featured projects like Dashboard and E-commerce designs.
- **Dark Mode Support**: Styled for both light and dark themes.

## Tech Stack

- **Framework**: Next.js 15
- **UI Library**: React 18
- **Styling**: Tailwind CSS
- **Components/Libraries**:
  - `react-fast-marquee`
  - `react-icons`
  - `react-slick` & `slick-carousel`
  - `swiper`
  - `react-toastify`

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
